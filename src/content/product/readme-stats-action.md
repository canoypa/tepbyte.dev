[GitHub Readme Stats](https://github.com/anuraghazra/github-readme-stats) を使用していましたが、プライベートリポジトリで使用している言語も含めてランキング表示したいなーということと、画像だと少し浮く感じがあったのでテキストで表示したいなー、ということで開発したものです。

集計方法が少し違うのか、コミット数などは少し減ってしまいました。
